# CS Patient Followup Forms (`cs_patient_followup_forms`)

Modulo para crear plantillas de seguimiento clinico totalmente configurables y rellenarlas por paciente.

## Funcionalidad base incluida

- Plantillas de seguimiento con secciones.
- Campos dinamicos por seccion (escala, si/no, texto, fecha, seleccion).
- Evaluaciones por paciente y profesional.
- Respuestas tipadas por campo.
- Smart button en contacto para ver evaluaciones del paciente.

## Roles

- Followup Admin
- Followup Clinician
- Followup Readonly
